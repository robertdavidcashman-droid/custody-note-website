# Website screenshots — NO CLIENT DATA

Screenshots in this folder are displayed on the public Custody Note website.

## Critical: Confidentiality

**Do NOT use screenshots that contain any real client information.**

Screenshots must not include:
- Client names
- UFN or reference numbers
- Real police station names (unless anonymised)
- Offence details
- Dates that could identify individuals
- Any other personally identifiable information (PII)

## How to capture safe screenshots

1. **Use a blank database** — Run Custody Note with the "Prepare trial" option to create a fresh database with no records, or use a copy with all data cleared.
2. **Use placeholder text only** — If forms are visible, use generic placeholders: "Client Name", "Example Station", "Sample offence".
3. **Verify before adding** — Check every screenshot for any real data before saving.
